# Ableton Live session notes

## Session structure

The demo session uses:

- one audio or sampler track routed through the ToucheSpatializer device
- Touché SE connected as MIDI controller

## Macro configuration

The device controls four macros mapped on the sampler track:

| Macro | Target parameter(s) | Range | Touché axis |
|---|---|---|---|
| Rack 1 | Gain, Low-Pass Filter, Reverb | 0 – 10% | CC16 pressure bottom |
| Rack 2 | Gain, Low-Pass Filter, Reverb | 10% - 100% | CC17 pressure top |
| Utility 1 | Panning | 50% -   0% | CC18 pressure left |
| Utility 2 | Panning | 50% – 100% | CC19 pressure right |

## Track chain (in order)

1. ToucheSpatializer.amxd
2. Sampler
3. Macro 1
4. Macro 2
5. Utility 1 
6. Utility 2

4. Return track A: Reverb (Hall or Plate)

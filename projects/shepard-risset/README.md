# shepard-risset

Interactive implementation of the Shepard-Risset glissando —
an auditory illusion of an endlessly rising or falling pitch.

## Content

- `shepard_risset_glissando.ipynb` : Jupyter notebook with
  real-time parameter control and audio synthesis
- `assets/audio/` : exported audio examples

## Dependencies

pip install numpy scipy sounddevice jupyter
